## Traveling Gallery App
This app based on React JS. (Sending Http Requests (e.g. Connecting to a Database), with backend code);


## Install Frontend
```sh
npm install
```

```sh
npm dev
```

## Install Backend
```sh
cd backend
```

```sh
npm install
```

``` sh
node app.js
```
